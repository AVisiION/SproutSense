SPROUTSENSE - PROJECT CONTEXT (SHORT)
=====================================

Project: SproutSense
Type: IoT Smart Plant Care
Status: Active, simplified documentation

CORE GOAL
---------
Monitor plant health and automate watering with ESP32 devices, backend, and dashboard.

MAIN PARTS
----------
1) ESP32 Sensor Controller
- Reads moisture, pH, temperature, humidity, light
- Controls pump through relay
- Sends sensor data to backend
- Shows live data on ST7735R TFT display

2) ESP32-CAM AI Module
- Captures plant images
- Runs disease detection workflow
- Sends disease events to backend

3) Backend (Node.js + Express + MongoDB)
- Stores sensor and watering history
- Exposes REST APIs
- Streams live updates to frontend

4) Frontend (React)
- Real-time dashboard
- Config and watering controls
- History and AI sections
